## 📐 Usage Guidelines

### Tailwind CSS Classes

```css
/* 背景色 */
bg-suzuka-50  /* 最も薄い背景 */
bg-suzuka-100 /* カードの背景 */
bg-suzuka-500 /* プライマリボタン */

/* テキスト色 */
text-suzuka-600 /* リンクテキスト */
text-suzuka-800 /* 本文テキスト */
text-suzuka-900 /* 見出しテキスト */

/* ボーダー色 */
border-suzuka-200 /* 通常のボーダー */
border-suzuka-300 /* ホバー時のボーダー */

/* リング色 */
ring-suzuka-500 /* フォーカスリング */
```

### グラデーション

```css
/* Linear Gradient */
.suzuka-gradient {
  background: linear-gradient(135deg, #ff9ebf 0%, #ff4785 100%);
}

/* Subtle Gradient */
.suzuka-gradient-subtle {
  background: linear-gradient(135deg, #ffe0ed 0%, #ffc2d9 100%);
}

/* Radial Gradient */
.suzuka-gradient-radial {
  background: radial-gradient(circle, #ff4785 0%, #e0266e 100%);
}
```

## 🎯 Best Practices

### Do's ✅

- **プライマリアクション**: `suzuka-500` を使用
- **ホバー状態**: 1段階濃い色を使用 (500 → 600)
- **無効状態**: 透明度を下げて表現 (`opacity-50`)
- **テキストコントラスト**: WCAG AA基準を満たす組み合わせを使用

### Don'ts ❌

- 隣接する色を同時に使用しない（例: suzuka-500 と suzuka-400）
- 薄い背景に薄いテキストを配置しない
- グラデーションを過度に使用しない

## 🌗 Dark Mode Support

ダークモードでは自動的に色が調整されます：

```css
/* Light Mode */
bg-suzuka-500 text-white

/* Dark Mode */
dark:bg-suzuka-400 dark:text-suzuka-950
```
