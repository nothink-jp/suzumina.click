Tailwind CSS のスペーシングスケールをそのまま採用しています。4 の倍数を基準にすることで、要素配置に視覚的なリズムを与えます。

## 階層的スペーシング

- **関連要素間**: `gap-1` / `gap-2` （4-8px）
- **同一グループ内**: `gap-3` / `gap-4` （12-16px）
- **セクション内**: `gap-6` / `gap-8` （24-32px）
- **ページレベル**: `gap-12` / `gap-16` 以上（48px〜）

## 用途別の使用例

| クラス | 値    | 用途                              |
| ------ | ----- | --------------------------------- |
| `p-2`  | 8px   | 小さなボタン、バッジ              |
| `p-3`  | 12px  | 標準ボタン                        |
| `p-4`  | 16px  | カード内コンテンツ                |
| `p-6`  | 24px  | 大きなカード・ダイアログ          |
| `p-8`  | 32px  | ページコンテナ                    |
| `mb-2` | 8px   | 関連テキスト間                    |
| `mb-4` | 16px  | 段落間・標準分離                  |
| `mb-6` | 24px  | セクション内要素分離              |
| `mb-8` | 32px  | セクション間分離                  |

## レスポンシブスペーシング

画面サイズに応じて余白を段階的に増やします。

```tsx
<div className="p-4 md:p-6 lg:p-8">
  {/* モバイル: 16px → タブレット: 24px → デスクトップ: 32px */}
</div>

<div className="space-y-4 sm:space-y-6 lg:space-y-8">
  {/* 要素間隔も段階的に変化 */}
</div>
```

### ブレークポイント別推奨値

| 画面サイズ          | コンテナ | セクション間 | 要素間 |
| ------------------- | -------- | ------------ | ------ |
| Mobile (~640px)     | `p-4`    | `mb-8`       | `mb-4` |
| Tablet (640px~)     | `p-6`    | `mb-12`      | `mb-6` |
| Desktop (1024px~)   | `p-8`    | `mb-16`      | `mb-8` |
