## Border Radius

<div style={{ display: "flex", gap: 24, flexWrap: "wrap", marginBottom: 24 }}>
  {[
    { name: "rounded-none", radius: "0" },
    { name: "rounded-sm", radius: "var(--radius-sm)" },
    { name: "rounded-md", radius: "var(--radius-md)" },
    { name: "rounded-lg", radius: "var(--radius-lg)" },
    { name: "rounded-xl", radius: "var(--radius-xl)" },
    { name: "rounded-2xl", radius: "16px" },
    { name: "rounded-3xl", radius: "24px" },
    { name: "rounded-full", radius: "9999px" },
  ].map((r) => (
    <div key={r.name} style={{ textAlign: "center" }}>
      <div
        style={{
          width: 80,
          height: 60,
          background: "hsl(var(--suzuka-200))",
          border: "2px solid hsl(var(--suzuka-500))",
          borderRadius: r.radius,
          marginBottom: 8,
        }}
      />
      <code style={{ fontSize: 12 }}>{r.name}</code>
    </div>
  ))}
</div>

角丸トークンは `globals.css` の `@theme`（`--radius: 0.5rem` 基準）から算出される v4 実効値です:
`rounded-sm = calc(--radius - 4px) = 4px`、`rounded-md = 6px`、`rounded-lg = --radius = 8px`、
`rounded-xl = calc(--radius + 4px) = 12px`。`rounded-2xl`(16px) / `rounded-3xl`(24px) は Tailwind v4 既定値。
コンポーネントの既定は `rounded-md`〜`rounded-lg`。

## 階層ガイドライン

影は UI の階層（z-index）を表現する手がかりです。

| レベル              | 影クラス      | 用途                       |
| ------------------- | ------------- | -------------------------- |
| Level 1 - Card      | `shadow-sm`   | 基本カード                 |
| Level 2 - Hover     | `shadow-md`   | カードのホバー浮上効果     |
| Level 3 - Dropdown  | `shadow-lg`   | ドロップダウン、ポップオーバー |
| Level 4 - Modal     | `shadow-2xl`  | モーダルダイアログ         |

## ボーダーカラー

境界線は **semantic トークン優先**。システムの境界色は暖色ニュートラル（`--border: 20 6% 90%`）で、
冷色系の `gray-*` / `red-*` / `green-*` の直書きは使いません（成功色は本システムにトークンが無い）。

| 用途             | クラス例（semantic 優先）                               |
| ---------------- | ------------------------------------------------------- |
| デフォルト境界   | `border-border`（暖色ニュートラル 20 6% 90%）           |
| 入力フィールド   | `border-input`                                          |
| ブランド境界     | `border-suzuka-200` / `border-suzuka-300`               |
| アクティブ       | `border-suzuka-500`                                     |
| フォーカス       | `ring-ring`                                             |
| エラー           | `border-destructive`                                    |

## 組み合わせ例

```tsx
// 標準カード
<div className="border border-border rounded-lg p-6 bg-card shadow-sm">…</div>

// 強調カード
<div className="border-2 border-suzuka-200 rounded-lg p-6 bg-suzuka-50 shadow-md">…</div>

// プレミアムカード
<div className="border border-suzuka-300 rounded-xl p-6 bg-card shadow-lg">…</div>

// ホバーインタラクション
<div className="shadow-sm hover:shadow-md transition-shadow duration-200 …">…</div>
```
