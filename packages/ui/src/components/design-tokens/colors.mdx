すべて WCAG 2.1 AA 準拠のコントラスト比を満たす組み合わせを想定して設計されています。

## 使用ガイドライン

| ロール          | クラス例                                          | 用途                                  |
| --------------- | ------------------------------------------------- | ------------------------------------- |
| Primary Action  | `bg-suzuka-500` / `hover:bg-suzuka-600`           | メインCTA、フォーム送信               |
| Secondary       | `bg-suzuka-100 text-suzuka-700`                   | 補助アクション、選択中の状態          |
| Outline         | `border-suzuka-300 text-suzuka-600`               | 控えめなアクション                    |
| Brand Card      | `bg-suzuka-50 border-suzuka-200`                  | ブランド色のカード背景                |
| Body Text       | `text-suzuka-800` / `text-suzuka-900`             | 本文・見出しテキスト                  |
| Accent          | `bg-minase-500` / `hover:bg-minase-600`           | サブCTA、アクセント要素               |
| Border (default) | `border-suzuka-200`                              | カード境界                            |
| Focus Ring      | `ring-suzuka-500`                                 | フォーカス表示                        |

## アクセシビリティ

- **本文 + 背景**: `bg-suzuka-50` + `text-suzuka-900`（コントラスト比 12.5:1）✓
- **Primary**: `bg-suzuka-500` + `text-white`（4.5:1 以上）✓
- **Danger**: `bg-suzuka-600` + `text-white`（5.0:1 以上）✓
- 隣接した明度（`suzuka-200` + `suzuka-400` など）の組み合わせは避ける
