## サイズスケール

| クラス        | サイズ | 用途                                       |
| ------------- | ------ | ------------------------------------------ |
| `h-3 w-3`     | 12px   | 統計情報、タグ内の小アイコン               |
| `h-4 w-4`     | 16px   | 標準的なボタン内アイコン（最頻出）         |
| `h-5 w-5`     | 20px   | 検索バー、中サイズの UI 要素               |
| `h-6 w-6`     | 24px   | 大きなボタン、ヘッダー要素                 |
| `h-8 w-8`     | 32px   | 強調表示、エラー状態                       |
| `h-12 w-12`   | 48px   | 空状態、大きな表示エリア                   |

## ベストプラクティス

```tsx
// 装飾的アイコン（テキストが意味を持つ場合）
<Play className="h-4 w-4" aria-hidden="true" />

// 意味のあるアイコン（単独で意味を持つ場合）
<Search className="h-4 w-4" aria-label="検索" />

// ボタン内アイコン
<button aria-label="動画を再生">
  <Play className="h-4 w-4" />
</button>
```

- **個別 import** で tree-shaking を効かせる（`import * as Icons` は禁止）
- **同機能には同アイコン**を徹底し、視覚的一貫性を保つ
- **タッチターゲット**は最小 44×44px を確保
